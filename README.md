## 🚀 Features

<details>
  <summary><strong>🩻 Symptom Checker</strong></summary>
  <ul>
    <li>Random Forest–based prediction from symptoms</li>
    <li>Confidence scoring with UI color indicators</li>
    <li>Mapped to medications, diets, precautions</li>
  </ul>
</details>

<details>
  <summary><strong>📄 Health Report Generator</strong></summary>
  <ul>
    <li>Generates PDF with logo, stamp, disease info</li>
    <li>Download & save to “My Reports” dashboard</li>
  </ul>
</details>

<details>
  <summary><strong>📂 Report Analyzer</strong></summary>
  <ul>
    <li>Upload PDFs or images of medical reports</li>
    <li>Analyzes using Groq (Claude/GPT-like LLM)</li>
    <li>Saves explanation to “Analyzer History”</li>
  </ul>
</details>

<details>
  <summary><strong>🔔 Medication Reminders</strong></summary>
  <ul>
    <li>Schedule pill reminders</li>
    <li>Frontend PWA support (coming soon)</li>
  </ul>
</details>

<details>
  <summary><strong>🛡️ Secure Auth</strong></summary>
  <ul>
    <li>JWT-based login/signup</li>
    <li>Protected backend API access</li>
  </ul>
</details>
